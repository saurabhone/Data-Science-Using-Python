Data Cleaning

The data is not always clean so it is necessary to clean the data before you analyze it.¶

There are multiple data problems which can cause data to be messy and make it difficult to analyze.¶

The data used in this exercise has common data problems such as¶

1. Inconsistent column names¶

2. Data type of columns¶

3. Missing Data¶

4. Untidy Data¶
