Seaborn: Python's Statistical Data Visualization Library¶

One of the best but also more challenging way to get our insights across data is to visualize it.¶

When we are using Python for data science, we’ll most probably use Matplotlib, a 2D plotting library that allows us to create publication-quality figures.¶

Another complimentary package that is based on this data visualization library is Seaborn, which provides a high-level interface to draw statistical graphics.¶

In this exercise we will plot graphs using the baseball data from¶

https://www.baseball-reference.com/teams/BOS/2016-schedule-scores.shtml#team_schedule::none

The observations in the above table are in the string data type and to plotting a graph using the same data type will give us an error as¶

TypeError: unsupported operand type(s) for /: 'str' and 'int'

In order to avoid that error, it is necessary that before plotting a graph we change it's data type to integer.¶

Moreover making the changes to original data type is also not advisable therfore we will create a copy of the data 'games' as 'games1' and make the changes in data type.¶
